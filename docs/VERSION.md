## 1. 현재 버전

- 현재 버전: `v0.9.0`
- 상태: Public MVP / Beta
- 기준일: 2026-05-13
- 단계: MVP 출시 완료 후 사용자 모집/홍보 운영

`v0.9.0`은 실제 프로덕션 배포와 핵심 사용자 흐름 검증이 끝난 공개 MVP 버전입니다. 아직 정식 `v1.0.0`으로 보지 않는 이유는 운영 관측성, 법무/동의 보강, 운영 문서화, 첫 사용자 피드백 반영이 남아 있기 때문입니다.

## 2. 버전 정책

이 프로젝트는 SemVer를 가볍게 따르되, 제품 단계에 맞춰 아래 기준으로 운영합니다.

- `v0.x`: MVP, 베타, 사용자 피드백 반영 단계
- `v0.9.x`: 공개 MVP 이후 버그 수정, 운영 안정화, 작은 UX 개선
- `v0.10.x`: 공개 MVP 이후 의미 있는 기능 추가 또는 주요 UX 개선
- `v1.0.0`: 정식 출시 기준 충족

변경 단위:
- patch: 버그 수정, 문구 수정, 작은 UI/UX 개선, 운영 설정 보강
- minor: 새 기능 추가, 주요 사용자 흐름 개선, 운영 기능 추가
- major: 큰 제품 방향 변경, 데이터 구조/호환성에 영향이 큰 변경

## 3. v1.0.0 기준

아래 항목이 충분히 정리되면 `v1.0.0` 정식 출시로 올릴 수 있습니다.

- 실제 사용자 피드백 1차 반영
- Vercel/Supabase/KBO 동기화 운영 로그 안정 확인
- 이용약관/개인정보처리방침 법무 검토
- 회원가입 또는 가입 전환 시 동의 체크박스 도입
- 환경변수와 seed/SQL 적용 순서 문서화
- README 실행/배포/운영 기준 보강
- 치명적인 온보딩, 직관 등록, 후기 작성, 공유 흐름 이슈 없음

Public MVP / Beta.

포함 기능:
- 랜딩, 로그인, 온보딩
- Google, Kakao, 이메일, 익명 로그인

- 직관 등록과 티켓 인증

- 경기 종료 결과 확인 모달
- 후기 작성, 후기 상세, 사진 캐러셀, 라이트박스
- 좋아요, 저장, 댓글

운영 백로그:
